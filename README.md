# CurrencyConverter

Консольное приложение для конвертации валют на основе официальных курсов ЦБ РФ.  
Проект разработан с использованием принципов модульности, разделения ответственности и Dependency Injection.  
Включает автотесты на xUnit и пример XML‑данных для тестирования парсера.

---

## Возможности

- Загрузка курсов валют с сайта ЦБ РФ
- Кэширование данных по датам
- Конвертация:
  - RUB → любая валюта
  - любая валюта → RUB
  - валюта → валюта (через рубли)
- Постраничный вывод списка валют
- Обработка ошибок и некорректного ввода
- Автоматические тесты (xUnit)

---

## Архитектура проекта

Проект построен по принципу разделения ответственности (Separation of Concerns) и разделён на три основных слоя:  
**инфраструктура**, **логика работы с данными**, **пользовательский интерфейс**.


CurrencyConverter.sln
│
├── CurrencyConverter/                 # Основной проект
│   │
│   ├── Program.cs                     # Точка входа
│   ├── AppServices.cs                 # DI-контейнер, регистрация сервисов
│   ├── HttpClientProvider.cs          # Обёртка над HttpClient
│   │
│   ├── Currency/                      # Логика работы с данными
│   │   ├── DataLoader.cs              # Загрузка и парсинг XML ЦБ РФ
│   │   ├── CurrencyService.cs         # Кэширование и предоставление курсов
│   │   ├── ConverterService.cs        # Математика конвертации валют
│   │   ├── MemoryCacheService.cs      # Кэш курсов по датам
│   │
│   ├── UI/                            # Консольный интерфейс
│   │   ├── MenuController.cs          # Главное меню и навигация
│   │   ├── ConsoleMessageService.cs   # Вывод сообщений и ошибок
│   │   ├── PaginateService.cs         # Постраничный вывод валют
│   │
│   ├── ValCurs.xsd                    # XSD-схема ЦБ РФ
│   ├── ValCurs.Designer.cs            # Автогенерируемые классы для XML
│
└── CurrencyConverter.Tests/           # Проект автотестов (xUnit)
    ├── ConverterTests.cs              # Тесты конвертации
    ├── DataLoaderParserTests.cs       # Тесты парсера XML
    └── TestData/
        └── sample.xml                 # Тестовый XML для парсера

### 1. Инфраструктура
- Program.cs — запуск приложения.
- AppServices.cs — DI-контейнер, создаёт и связывает сервисы.
- HttpClientProvider.cs — единая точка для HTTP-запросов.

### 2. Логика работы с данными (Currency/)
- DataLoader — загрузка XML, десериализация, обработка ошибок.
- CurrencyService — кэширование, выдача курсов, работа с датами.
- ConverterService — математическая конвертация валют.
- MemoryCacheService — хранение курсов по датам.

### 3. Пользовательский интерфейс (UI/)
- MenuController — логика меню, обработка выбора пользователя.
- ConsoleMessageService — вывод сообщений, ошибок, подсказок.
- PaginateService — постраничный вывод списка валют.

### 4. Автотесты (CurrencyConverter.Tests/)
- xUnit‑тесты для ключевых компонентов.
- Тестовые XML‑файлы для проверки парсера.
